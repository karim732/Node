- Error handling middleware.

Http status code.

2xx (Success)
 - 200 - operation succeeded.
 - 201 - Success, resource created(backend) 

3xx (Redirection)
 - 301 - moved permanently

4xx (Client side error)
 - 401 - Not authenticated
 - 403 - Not authorized
 - 404 - Page not found
 - 422 - Invalid input

5xx (server side error)
 - 500 - Server side error
